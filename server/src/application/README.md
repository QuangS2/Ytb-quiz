# Application Layer

Tầng Logic Ứng Dụng. Tầng này điều phối các thực thể domain để thực hiện các nghiệp vụ cụ thể.

## Nội dung chính:
- **Use Cases**: Lớp thực thi logic nghiệp vụ cụ thể (ví dụ: `CreateQuizUseCase`).
- **Ports**: Các interface định nghĩa hợp đồng đầu ra và đầu vào (ví dụ: `QuizRepositoryPort`).
- **DTOs**: Các đối tượng vận chuyển dữ liệu qua lại giữa Presentation và Application.
