# Components Folder

Thư mục chứa các UI Component dùng chung toàn bộ dự án.

## Cấu trúc:
- `ui/`: Các Component nguyên tử (Atomic components) như:
  - `Button/`
  - `Input/`
  - `Card/`
  - `Modal/`
  - `Badge/`
  - `Toast/`
- `layout/`: Các component bố cục khung giao diện như:
  - `AppShell/`
  - `Sidebar/`
  - `Topbar/`
